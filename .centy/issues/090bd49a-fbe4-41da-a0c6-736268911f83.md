# Pre-commit hook: lint with cargo clippy

## Overview

Add a pre-commit hook that runs `cargo clippy` to catch common Rust mistakes and style issues.

## Implementation

Add to `.husky/pre-commit`:

```sh
cargo clippy --all-targets --all-features -- -D warnings
```

## Notes

- `-D warnings` treats all clippy warnings as errors, blocking the commit
- Uses the pinned nightly toolchain from `rust-toolchain.toml`
- This is the same lint tooling recommended for Dylint lint development

## Acceptance Criteria

- Hook blocks commit when clippy emits any warning or error
- Hook passes on clean code

# Pre-commit hook: format check with cargo fmt

## Overview

Add a pre-commit hook that ensures all Rust code is formatted with `rustfmt`.

## Implementation

Add to `.husky/pre-commit`:

```sh
cargo fmt --all --check
```

## Notes

- Uses the pinned nightly toolchain defined in `rust-toolchain.toml`
- `--check` mode exits non-zero if any file would be reformatted (does NOT modify files)
- Developers should run `cargo fmt` manually before committing

## Acceptance Criteria

- Hook blocks commit when code is not formatted
- Hook passes on correctly formatted code
